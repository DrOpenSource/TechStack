## üéì Usage Example

```

"Use product-architect to create user stories for user registration"

# 2. Design architecture
"Use system-architect to design authentication system with OTP"

# 3. Create prototype (optional)

"Use design-qa to validate the login prototype"

# 5. Implement backend

# 6. Implement frontend
"Use frontend-dev to build login UI components"

"Use qa-engineer to create test suite for authentication"

# 8. Deploy
"Use devops-agent to deploy to Vercel"
```
